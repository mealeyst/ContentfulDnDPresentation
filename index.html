In the sake of brevity, I am only going to give a high level overview of how Dungeons & Dragons work. The Dungeon Master and his group of players work together to create a narrative whose outcomes are determined by rolling dice and referencing what is called a character sheet. The character sheet is one of the most important parts of the game itself. It’s a reference sheet that indicates how well a character can perform a certain action. This can be anything from convincing a guard to let players pass without paying a toll, to what spells a player can cast, to what equipment and wealth a player has.

The character sheet is a rather complex.

How does one get started?

Using the Contentful Extension CLI Tool, it's only 7 steps!

We are going to the terminal

Step 1: Install the Create Contentful Extension CLI Tool 
              
                $npm install create-contentful-extension;
                $npx @contentful/create-contentful-extension example-extension
Step 2: Select the Field extension type. 
                
                  ? Select type of extension: (Use arrow keys)
                  ❯ Field extension
                    Sidebar extension
Users need the following:
  • Node 8 or higher
  • NPM 5.2 and higher
Step 3: Select the field type that you are creating the interface for (Object in our case). 
              
                ? Select field types for this extension:
                 ◯ Number
                 ◯ Date
                 ◯ Boolean
                ❯◉ Object
                 ◯ Entry
                 ◯ Entries
                 ◯ Asset
Step 4: Select your JavaScript flavor! 
               
                 ? What language you want to use to develop extension: (Use arrow keys)
                 ❯ JavaScript
                   TypeScript
Step 5: Install your dependencies: 
              
                npm install
Step 6: Be sure to login to your contentful instance and add your spaceID as well as environmentID 
                
                  npm run login && npm run configure
Step 7: Start the server and start developing! 
              
                npm start
When you have finished your development deployment is a breeze 
              
                npm run deploy
Running this command compiles your extension into a neat little index.html file which then gets pushed up to Contenful.
